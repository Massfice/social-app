This directory contains all system files/classes, so please, don't touch this files, because system could collapse.

Contains:

a) cCleanUrl class - class for handling clean urls.
b) cSocialApp class - starting class of application.
c) cRoleManager - class for handling roles.
d) cSmarty - class for handling Smarty
e) cOkta - class for handling Okta/Oauth2
